Документация для базы данных друзей
Описание проекта
Цель: Разработка базы данных для хранения информации о друзьях с возможностью добавления, редактирования, удаления, поиска и фильтрации данных.

Создать базу данных для хранения информации о друзьях.
База данных должна содержать следующие поля: ФИО друга, номер телефона, электронная почта, фотография, краткое описание. 
Проект выполним в течение 6 недель командой из 6 человек.
Завершение разработки за 6 недель.
Требования к проекту
Функциональные требования:
1.	Добавление нового друга:
•	Возможность указать:
•	Полное имя (ФИО).
•	Номер телефона.
•	Электронную почту.
•	Дату рождения.
•	Фотографию.
•	Краткое описание (например, интересы или заметки).
2.	Редактирование информации о друге:
•	Изменение любых полей (например, обновление номера телефона или описания).
3.	Удаление друга из списка.
